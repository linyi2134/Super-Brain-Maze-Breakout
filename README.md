# Super-Brain-Maze-Breakout
本项目基于综艺节目《最强大脑》第十三季“密阵突围”赛题规则，使用 C++ 面向对象思想实现一款策略逻辑类桌面游戏。程序以控制台界面为基础，完成棋盘生成、棋子联动移动、代码合并、得分计算、游戏胜负判定等核心逻辑，并通过文件操作实现游戏数据持久化、排行榜记录。  主要实现功能： 1.支持10×10、25×25等多种规格棋盘，自动生成随机代码盘面。 2.实现棋子联动移动、空白格无限移动、代码合并、吃子、得分与奖励分机制。 3.支持单人模式、双人对战模式，可进行完整对局流程。 4.支持判断操作正误，并给出对应的提示和盘面变化 5.实现游戏回合控制、跳过判定、游戏结束与胜负判定逻辑。 6.通过文件保存玩家信息、对局记录、得分、用时，生成排行榜。

密阵突围规则
一、游戏基础信息
支持模式：单人模式 / 双人对战 / AI 对战
代码组成：字母 A–E + 数字 0–9 组合（如 A3、B7、D9）
核心目标：通过合并与吃子积累分数，总分更高者获胜；同分则用时更短者胜

二、棋盘规格与初始棋子数（严格按你 Word 原文）
5×5 棋盘 → 每人 1 个初始代码
10×10 棋盘 → 每人 2 个初始代码
15×15 棋盘 → 每人 3 个初始代码
20×20 棋盘 → 每人 4 个初始代码
25×25 棋盘 → 每人 5 个初始代码

三、初始设定
棋盘所有格子随机生成代码
玩家按当前棋盘规格，选择对应数量的初始代码棋子
对战模式下，双方棋子区分阵营，不混淆

四、核心移动规则（联动移动）
统一方向指令
每一步选择上 / 下 / 左 / 右，所有己方棋子同步执行
独立移动判断
每枚棋子单独检测是否可移动，能动则动，不能动则原地不动，互不影响
空白格规则
棋子离开原位置后，该格变为空白格
棋子在空白格内可无限步、任意方向、任意次数连续移动
空白格移动不触发合并、不得分、不结束回合
边界处理
移动超出棋盘边界 → 该棋子无法移动，留在原地

五、代码合并规则（你定制唯一版）
字母映射
A=1，B=2，C=3，D=4，E=5
字母相加后对 5 取余：
余 1 → A
余 2 → B
余 3 → C
余 4 → D
余 0 → E
数字规则
数字直接相加，结果对 10 取余（只保留个位）
合并类型
字母相同、数字不同
字母不变，数字相加取个位
例：A3+A4=A7；B9+B7=B6
数字相同、字母不同
数字不变，字母按五进制相加取余映射
例：D7+E7 = (4+5) mod5 = 4 → D7
字母数字完全相同
合二为一，代码不变
例：A1+A1=A1
字母数字均不同
不可合并，无法移动至此格

六、吃子规则（敌方棋子）
敌方棋子视为普通代码，满足合并条件即可合并
合并成功：
己方 +1 基础分
敌方该棋子永久移除
原格变为空白格
合并触发 → 本回合立即结束

七、计分规则（方案 A・强策略防无脑冲脸）
基础得分
合并中立代码：+1 分
吃掉对方 1 枚棋子：+1 分
奖励得分
单回合吃掉对方 3 枚：额外 +5 分
单回合吃掉对方 5 枚：额外 +10 分
全歼对方所有棋子：额外 +20 分

八、回合规则
一回合定义
玩家一次：选择方向 → 移动 → 停止 / 合并 的完整操作
回合结束条件（满足其一即结束）
成功合并（中立代码 / 敌方棋子）
在空白格移动后主动停止
所有棋子均无法移动
跳过判定（满足其一即为跳过）
本回合未发生任何合并
仅在空白格移动后停止
无任何棋子可移动

九、游戏结束条件
满足任意一条，游戏立即结束：
一方所有棋子被全部吃掉
任意一方连续跳过 2 个回合
达到设定的回合上限 / 时间上限

十、胜负最终判定（无任何漏洞）
优先比较总得分，分数高获胜
分数相同 → 比较总用时，用时短获胜
被吃光 ≠ 直接判负，一切以分数为准
十一、补充稳定规则
同一回合内可在空白格无限穿梭，直到主动停止或触发合并
合并一旦发生，立刻停止所有移动，回合结束
单人模式：无对手，目标在限定条件内冲击更高分数
对战模式：双方轮流操作，按回合顺序进行
